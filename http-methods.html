Exemplo 1: GET

Essa página que você está lendo foi carregada através do protocolo HTTP. O que aconteceu quando você digitou a url do desse site no navegador foi mais ou menos o seguinte:


O navegador pegou o endereço desse post, que é http://marcobonuto.com/workshop/http-methods.html. Em seguida, ele montou uma requisição HTTP da seguinte forma(esse texto mesmo):


GET /http/ HTTP/1.1 Host: marcobonuto.com


Aí ele pegou esse texto e enviou pro servidor onde meu site está hospedado. O servidor do meu site interpretou essa requisição, viu o que era necessário, e devolveu uma resposta para o navegador num formato que ele entende: html.
